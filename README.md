# retail-sales-forecasting
End-to-end sales analytics pipeline for a multi-product coffee chain: revenue forecasting, demand pattern analysis and inventory recommendations 
built in Python.

## Retail Sales Forecasting & Business Intelligence

## Overview
- Engineered data cleaning and transformation workflows using pandas and NumPy
- Modeled revenue trends and seasonal demand patterns across time
- Analyzed hourly demand to identify peak and off-peak sales windows
- Evaluated product-level performance across the full menu
- Delivered ranked recommendations for inventory optimization and 
  peak-hour resource allocation

## Key Outcomes
- Identified highest revenue-generating products and peak demand hours
- Quantified inventory inefficiencies previously managed by guesswork
- Produced actionable staffing and stocking recommendations backed by data

## Tools Used
- Python — pandas, NumPy, Matplotlib, Seaborn
- Jupyter Notebook
